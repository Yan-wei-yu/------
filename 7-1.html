Features

Feature 1. Open Printer / Resin management

Convenient printer/resin/layer thickness management function to let user edit full printer parameters.

Feature 2. Support Genearation & Edit

There are standard and structure type of support and allow full support shape edit after support already build.

Feature 3. Hollow & Drill Hole

Two types of hollow are support, one is 3D equal thickness,the other is penetrating at one plane to save resin usage.

Feature 4. Label

Add texts with size control for inward and outward label.

Feature 5. Slice & Output

Fast and robust slicing function can handle take care quality problem. Image file and slc file are both support for different application.

Feature 6. Printer Connection

Allowing search and save printers from intranet,upload file to printer and start printing task.

Feature 7. Orthodontic Study Models Build Printing Model
  • Auto rotate model and detect boundary.
  • Manual edit base height and shape of boundary.
  • Fast trim and build final printing model.
Feature 8. Digital Intraoral Scanner Build Printing Model
  • Auto rotate and detect boundary.
  • Manual edit base height and shape of boundary.
  • Fast trim and build final printing model.
Feature 9. Auto Rotate and Build
  • Auto determine model orientation and build supports.
  • Auto build text date at base of support.
  • Batch process all model at the same time.
Feature 10. Lattice structure

Lattice structure can be generated inside the model.
